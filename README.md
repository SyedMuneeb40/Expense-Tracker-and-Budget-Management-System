# Expense-Tracker-and-Budget-Management-System
A simple Expense Tracker web application built using ASP.NET Core MVC, Entity Framework Core, and SQL Server.

## Features

- Add Income Transactions
- Add Expense Transactions
- Edit Transactions
- Delete Transactions
- Dashboard Overview
- Category Management
- SQL Server Database Integration
- Entity Framework Core Migrations

## Technologies Used

- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQL Server
- Bootstrap
- HTML/CSS

## Project Structure

Controllers
Models
Views
wwwroot
Migrations
